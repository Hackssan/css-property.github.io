Common CSS Properties

Pertaining to Visual Styles:

Property Description
font-family Used to express the font desired

Syntax:

h1{ font-family: Helvetica; }

When two words the font should be declared in parentheis
Property Description
font-size Used to increase or decrease size of typography

Syntax:

h1{ font-size: 200px; }

Size can be set by percentage, ems and pixels
Property Description
font-weight Used to darken or lighten text font

Syntax:

h1{ font-weight: 400; }

Size can be set by numerical values where 400 is standard 700 is bold it can also take the values of: normal, bolder, lighter, bold
Property Description
text-align Aligns text to it's parent element

Syntax:

h1{ text-align: justify; }

this property takes one of four values: right, left, center and justify which spaces out text to align with the left and right of the parent element
Property Description
color sets foreground color
background-color sets background color

Syntax:

h1{ color: red; }

h1{ background-color: rgba(125,125,125,1); }

rgba allows us to experiment with specific shades of colors over naming specific colors and opacity with the alpha value
Property Description
opacity transparancy vs opaqueness

Syntax:

h1{ opacity: .5; }

While rgba allows us to modify color and opacity can't argue with more tools for specificity. 0 is translucent 1 is opaque
Property Description
background-image sets the image background to a url relative or absolute that you specify

Syntax:

h1{ background-image: url(); }

When specifying a absolute path "" are used around the path within the attribute url

Pertaining to the Box Method:

Property Description
height/width scale how tall or how wide an element is respectively.
padding adjust the space within the border and the element itself. think insulation.
border adjust the containing box around all elements. There are ways to change the shape to circular.
margin scale the distance between an element's box and surrounding elements

Syntax:

h1{ height: 200px; width: 100%; padding: 10px 20px 5px; margin: 5px 2px 8px 10px; border: 5px solid red }

height/width: can be set in pixels, ems and percentages. padding along with margin can be shorthanded. It's always represented clockwise. 3 numbers = top, left/right, bottom You can create a border that is a perfect circle by first creating an element with the same width and height, and then setting the border-radius equal to half the width of the box, which is 50%
Property Description
overflow controls content that spills or overflows out of it's box.

Syntax:

h1{ overflow: hidden; }

As it suggests, hidden hides any content that spills over. Then there's scroll which'll create a scrollbar in a browser to view the content spilt. (scroll) Finally hidden. Which hides all spilt over content. It's important to note that the element in question needs to have definite dimensions otherwise it'll keep expanding.
Property Description
visibility elements can be hidden from view with this property

Syntax:

h1{ visbility: hidden; }

Note in this example above the space reserved for the h1 example above is still present. However it's blank. It can also take the values: collapse, and visible which is the default input
